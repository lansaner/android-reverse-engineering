## 工具集

* [apktool][1]

    A tool for reverse engineering Android apk files

* [dex2jar][2]

    Tools to work with android .dex and java .class files

* [jd-gui][3]

    A standalone Java Decompiler GUI

* [Android Studio][4]

    用于动态调试。

* [IDA Pro][5]

    最强大的静态逆向分析工具，没有之一。

* [Smali 语法高亮与 Tag 支持][6]

    Vim 里的 Smali 语法支持。
